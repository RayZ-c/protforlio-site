## Overview {.outline-only}

<details class="section-panel">
  <summary class="section-header">
    <span class="section-title">✮ Overview</span>
    <span class="section-icon">▾</span>
  </summary>

<div class="project-text">

  Custom combat moveset built to client specification:

  - **4 Distinct Abilities:** Each with unique input, cooldown, animation, and particle feedback
  - **Transformation/Power-Up System:** Visual state change with stat multipliers
  - **Animations:** Custom Roblox rig animations for all abilities + transformation
  - **Visual Effects:** Particle systems synced to ability timing
  - **Sound Design:** Custom SFX for each ability + transformation activation
  - **Scripting:** Pure Lua backend (input handling, ability sequencing, cooldown management)

</div>
</details>
